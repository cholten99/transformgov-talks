How to export from Libsyn
  1. Log in to five.libsyn.com
  2. Click Stats in the top navigation
  3. Click the Download Report button (upper-right of the page)
  4. Tick Downloads by Episode (at minimum) then click Export
  5. A .zip file downloads — upload it here, or extract the CSV inside and upload that

How to export from Luma
  1. Log in to lu.ma
  2. Open the event you want to export
  3. Go to Guests → click Export (top-right)
  4. Download the CSV and upload it here — repeat for each event you want to add
  5. Note: uploading a new file replaces all previous Luma data
